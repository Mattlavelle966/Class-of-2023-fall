Flex Example 2

The flex items have flex:1. Each flex item (except yellow marten) is the same width and the space is filled. As the window gets smaller, there comes a point where that isn't feasible. Flex then tries to appropriately distribute the flex items.

The bigbox flex item has flex:2.75. This means bigbox is 2.75 times the size of the other flex items. As you resize smaller, the 2.75 isn't feasible and flex redistributes. The space is always filled.

Note that the browser is making the decision based on what it thinks is best for the current device.
