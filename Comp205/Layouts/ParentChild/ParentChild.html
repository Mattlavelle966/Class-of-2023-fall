Parent Child Example

This mainsection div element has an aqua background. It is inside the body element. The body element is its parent.

The width of this aqua div is set to 50%. This means it takes up 50% of the width of its parent. It takes up 50% of the width of the body element.

Indigenous Art

This div has a dark coral background. It is inside the mainsection div element. The mainsection div element is its parent.

The width of this div is set to 75%. This means it takes up 75% of the width of its parent. It takes up 75% of the width of the mainsection div element. It does not take up 75% of the width of the web page.
